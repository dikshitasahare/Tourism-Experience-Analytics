# 🌍 Tourism Experience Analytics: Classification, Prediction, and Recommendation System

Tourism recommendation web application built with **Flask** and **Machine Learning** that helps users discover personalized travel destinations based on their preferences. The application combines recommendation algorithms, interactive dashboards, destination analytics, and rich travel information to provide an engaging trip-planning experience.

## ✨ Features

* 🎯 Personalized destination recommendations using Machine Learning
* 🌎 Country Explorer with detailed destination information
* 📊 Interactive dashboards for tourism insights and analytics
* 🗓️ Best time to visit recommendations
* ⚖️ Destination comparison
* ⭐ Destination ratings and popularity analysis
* 🗺️ Interactive maps for location exploration
* 📱 Responsive and user-friendly web interface
* 📈 Data visualization for tourism trends
